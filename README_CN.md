# CS2Lab — CS2 道具实验室

> 每张地图、每种道具，都有图文并茂的教学。从站位到准星到投法，一看就会。

## 痛点在哪

CS2 里，一颗烟能决定一局的输赢。但靠谱的点位教程，不是要你在 YouTube 上反复拖进度条，就是要记一堆版本一更新就可能失效的参照物。

**CS2Lab** 把道具教学做成一站式工具：截图直接标好准星该瞄哪、投掷说明写得清清楚楚、道具怎么组合使用也有图解。

## 有什么功能

| 模块 | 里面有什么 |
|------|-----------|
| **道具图鉴** | 烟雾弹、闪光弹、燃烧弹、手雷的基础数据：伤害、范围、持续时间、价格 |
| **投掷教学** | 站位在哪、准星瞄哪（截图标注）、怎么扔（左键/右键/跳投） |
| **组合教学** | 先扔烟封视线，再扔闪突破——用地图叠加图展示覆盖范围 |
| **地图覆盖** | 竞技图池 7 张全做：Dust2、Mirage、Inferno、Nuke、Vertigo、Ancient、Anubis |
| **智能搜点** | 描述你遇到的场景（比如"Dust2 当 T 拉中路总被打"），自动推适合的道具 |
| **用户投稿** | 你有好点位？提交上来，审核通过就上架 |
| **收藏笔记** | 常用的点位收藏起来，自己加备注 |

## 地图覆盖

竞技图池全部 7 张，从 Dust2 开始一张一张做。

## 怎么做内容

- **不做 3D** — 用游戏官方地图截图，在上面标注
- **截图定位** — 准星瞄的地方画圈标出来，不用猜
- **图文都有** — 文字说明配标注截图，学到会为止

## 明确不做的

- ❌ 不搞战术分析 — 不给回放加标记，不生成队伍打法
- ❌ 不当 AI 教练 — 智能推荐是从已有教学里匹配，不是 AI 现场编
- ❌ 不做武器大全 — 聚焦投掷道具，不碰枪械

## 怎么用

### 线上

1. 打开 [oxelia51.com](https://oxelia51.com) 注册登录
2. 工具菜单里找到 CS2Lab
3. 看地图、搜点位、收藏你常的

### 桌面版

1. 去 [GitHub Releases](https://github.com/XiaoleC05/CS2Lab/releases) 下载 `CS2Lab.exe`
2. 双击打开
3. 数据打包好的，不联网也能看
