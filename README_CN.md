# CS2Lab — CS2 道具教学实验室

> 掌握每张地图的每一种道具。分步指南、瞄准截图、智能推荐。

## 为什么需要 CS2Lab？

在 CS2 中，一颗烟雾弹可能决定回合胜负。但要找到可靠的道具投掷点位，意味着你得反复看 YouTube 视频、逐帧暂停、记住那些随着版本更新就会变化的参照物。

**CS2Lab** 是你的一站式道具知识库：清晰的截图标注准星位置、详尽的投掷说明、道具组合配合指南——还在不同地图上灵活运用。

## 功能

| 模块 | 内容 |
|------|------|
| **道具图鉴** | 烟雾、闪光、燃烧、手雷的参数——伤害、半径、持续时间、价格 |
| **投掷教学** | 站位标记 + 准星参照截图 + 投掷方式（左键/右键/跳投） |
| **配合教学** | 文字说明 + 地图叠加展示多道具覆盖范围 |
| **地图覆盖** | 全部7张竞技地图：沙二、米拉吉、炼狱、核子、殒命、远古、阿努比斯 |
| **智能搜索** | 描述你的场景（如"沙二T方拉中路"）→ 推荐适用道具 |
| **社区投稿** | 提交你的点位，管理员审核后上架 |
| **收藏笔记** | 收藏常用点位，添加个人心得 |

## 内容风格

- **无 3D 渲染** — 使用官方地图截图加标注叠加层
- **截图定位** — 准星位置清晰标记
- **图文结合** — 每个点位同时提供文字说明和标注截图

## CS2Lab 不做什么

- ❌ 不做战术分析 — 不分析对局回放或制定队伍策略
- ❌ 不做 AI 教练 — 智能搜索是根据场景匹配已有教学，不是 AI 生成新内容
- ❌ 不做完整武器指南 — 仅聚焦投掷道具

## 使用方式

### 在线版

1. 访问 [oxelia51.com](https://oxelia51.com) 注册登录
2. 在工具菜单中打开 CS2Lab
3. 浏览地图、搜索点位、收藏常用

### 桌面版（exe）

1. 从 [GitHub Releases](https://github.com/XiaoleC05/CS2Lab/releases) 下载 `CS2Lab.exe`
2. 运行即启动本地界面
3. 数据打包在内，无需联网即可浏览
